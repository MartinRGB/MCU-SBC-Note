### Device Used

- Two way mirror
- Raspberry Pi CM4 with Waveshare Carrier Board

### Steps

Two Options:

- Compile [glslViewer](https://github.com/patriciogonzalezvivo/glslViewer/wiki) and run
- Compile [kms-glsl](https://github.com/astefanutti/kms-glsl) and run
